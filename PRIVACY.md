# Privacy Policy - LedgerTrace

**Effective Date:** November 5, 2025  
**Last Updated:** November 5, 2025

## Information We Collect

### Public Records Data
LedgerTrace aggregates and analyzes publicly available information from:
- Florida Division of Corporations (Sunbiz)
- County Property Appraiser records
- IRS Exempt Organization database
- Small Business Administration records

### User Input Data
When you use our service, we temporarily process:
- Entity names you search for
- Addresses you provide for property analysis
- No personal information about you is collected

### Technical Data
Our servers may log:
- IP addresses (for rate limiting and abuse prevention)
- Search queries (for service improvement)
- Usage timestamps (for analytics)

## How We Use Information

### Analysis Purpose
- Generate risk assessments based on public data patterns
- Provide source verification links
- Display property and business registration information

### Service Improvement
- Monitor usage patterns to improve accuracy
- Identify commonly searched entities for optimization
- Debug technical issues and improve performance

## Data Retention

### Search Data
- Search queries are not permanently stored
- Temporary processing data is deleted within 24 hours
- No user profiles or search history maintained

### Public Records
- We do not store copies of government records
- All data is fetched in real-time from official sources
- Links direct users to original government databases

## Data Sharing

### No Personal Data Sharing
- We do not collect personal information about users
- No user data is sold or shared with third parties
- Search queries are not linked to individual users

### Public Information Only
- All displayed information comes from public government records
- Users are encouraged to verify information through official sources
- Source URLs are provided for all data points

## Security Measures

### Technical Safeguards
- API rate limiting to prevent abuse
- Secure HTTPS connections for all communications
- Input validation to prevent malicious queries
- No storage of sensitive user information

### Transparency
- Open source codebase available for public review
- All data sources clearly identified and linked
- Methodology and scoring algorithms are publicly documented

## Your Rights

### Corrections
- If information about your organization appears incorrect, submit a correction request
- We will investigate and update our analysis methodology if needed
- All corrections are handled through our public GitHub repository

### Opt-Out
- Contact us to request exclusion of specific entities from analysis
- Requests must come from authorized representatives of the entity
- Exclusions will be publicly documented for transparency

## Legal Limitations

### Not Legal Advice
- LedgerTrace provides informational analysis only
- Results should not be used as sole basis for legal or financial decisions
- Consult qualified professionals for specific legal advice

### No Warranties
- Information provided "as is" without warranties of any kind
- We do not guarantee accuracy, completeness, or timeliness
- Users assume all risks associated with use of the service
